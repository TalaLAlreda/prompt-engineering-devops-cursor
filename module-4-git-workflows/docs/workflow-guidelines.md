# Workflow Guidelines

## Pull Requests
- Use appropriate PR template
- Title follows Conventional Commits
- Link issues (Fixes #123)
- Request reviewers via CODEOWNERS

## Reviews
- Min 2 approvals
- All required checks green
- No direct pushes to main/develop
